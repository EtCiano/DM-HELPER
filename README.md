O Dungeon Master Helper é uma ferramenta feita para ajudar mestres de RPG (DMs) na hora de mestrar um RPG, possúi ferramentas para contabilizar o tempo dos turnos, manter a organização da lista de iniciativa, converter moedas, fazer fichas de inimígos e NPCs entre mais ferramentas que estão por vim.
============================================================================================
Dungeon Master Helper is a tool designed to Dungeon Masters (DMs) when running an RPG. It has tools for tracking turn times, keeping the initiative list organized, converting currencies, creating enemy and NPC sheets, among other tools to come.
============================================================================================
